Endpoint 1 — Register & Get API Key
/wallet  —  Get API Key
GET
Visit the wallet page to generate your API key. Your key is shown once — save it securely. 
// Open in browser to get your API key
https://safeora.onrender.com/wallet
POST /api/register
POST
Register with your Telegram user ID to create an account and receive your API key.
FieldTypeDescription
userid required string Your Telegram user ID 
// Request Body
{
  "userid": "1054928757"
}
200 SUCCESS 
{
  "success":  true,
  "api_key":  "safeora-7f3d8e...",
  "message":  "API KEY GENERATED."
}
Endpoint 2 — Balance
GET /api/balance
GET
Returns current wallet balance in TON and nanoTON. Requires x-api-key header.
200 RESPONSE 
{
  "balance":      "12.500000",
  "balance_nano": 12500000000,
  "currency":     "TON"
}
Endpoint 3 & 4 — Deposit
GET /api/deposit/info
GET
Returns your unique deposit address and memo (comment) required for TON deposits.
200 RESPONSE 
{
  "address": "EQD...deposit",
  "memo":    "21049012",
  "network": "TON"
}
POST /api/deposit/verify
POST
Verify a TON deposit by submitting the transaction hash and amount.
FieldTypeDescription
tx_hash requiredstringTON transaction hash
amount requiredstringDeposit amount in TON
 
{
  "tx_hash": "e6d3f...",
  "amount":  "5.0"
}
200 RESPONSE 
{ "verified": true, "transaction": "hash_abc..." }
Endpoint 5 — Withdraw TON
POST /api/withdraw
POST
Send TON to any wallet address. Deducted from your SAFEORA balance instantly.
FieldTypeDescription
address requiredstringRecipient TON wallet address
amount requiredstringAmount in TON (e.g. "1.25")
 
// POST https://safeora.onrender.com/api/withdraw
{
  "address": "UQAbc123...DEF",
  "amount":  "1.25"
}
200 SUCCESS 
{
  "success": true,
  "tx_hash": "0x7f3c...",
  "message": "Withdrawal initiated"
}
4xx ERROR 
{ "success": false, "error": "Insufficient balance" }
Endpoint 6 — Transaction History
GET /api/transactions
GET
Retrieve paginated transaction history. Supports ?limit=20&offset=0 query params.
200 RESPONSE 
{
  "transactions": [
    {
      "tx_id":     "hash_01",
      "type":      "withdraw",
      "amount":    "0.5",
      "timestamp": 1700000000
    }
  ],
  "total": 42
}
Standard Responses
200 Success
{
  "success": true,
  "data":    {}
}
4xx / 5xx Error
{
  "success": false,
  "error":   "Invalid API key"
}
Code Example — Withdraw
JavaScript — Withdraw Request 
const response = await fetch(
  "https://safeora.onrender.com/api/withdraw",
  {
    method:  "POST",
    headers: {
      "x-api-key":     "safeora-7f3d8e...",
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      address: "UQXXXX...",
      amount:  "0.05"
    })
  }
);

const data = await response.json();
console.log(data.tx_hash);
