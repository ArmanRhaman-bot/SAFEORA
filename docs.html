Authentication & Headers
Request Headers
x-api-key:      YOUR_API_KEY
Content-Type:  application/json
Endpoint 1: Registration
POST /api/register
{
  "username": "alice",
  "email":    "alice@example.com"
}
✅ Response 200:
{
  "success": true,
  "api_key":  "sa_live_7f3d8e...",
  "message":   "Account created. Store your API key securely."
}
GET /api/balance
Returns current wallet balance (TON and nanoTON).
{
  "balance":     "12.500000",
  "balance_nano": 12500000000,
  "currency":     "TON"
}
GET /api/deposit/info
Get unique deposit address and memo (comment) for TON deposits.
{
  "address": "EQD...deposit",
  "memo":    "21049012",
  "network":   "TON"
}
POST /api/deposit/verify
{
  "tx_hash": "e6d3f...",
  "amount":   "5.0"
}
✅ Response:
{"verified":true, "transaction":"..."}
Endpoint 5: Withdraw TON
POST /api/withdraw
{
  "address": "UQAbc123...DEF",
  "amount":  "1.25"
}
✅ Success (200):
{
  "success":  true,
  "tx_hash":  "0x7f3c...",
  "message":  "Withdrawal initiated"
}
⚠️ Error (4xx):
{"success":false, "error":"Insufficient balance"}
GET /api/transactions
Transaction history with pagination (limit, offset).
{
  "transactions": [
    {
      "tx_id":     "hash_01",
      "type":      "withdraw",
      "amount":    "0.5",
      "timestamp": 1700000000
    }
  ],
  "total": 42
}
SAFEORA Code Example
SAFEORA Script (Withdraw)
HTTP.post({

  url:
    "https://tbl-earn-production.up.railway.app/api/withdraw",

  headers: {
    "x-api-key":      "YOUR_API_KEY",
    "Content-Type":  "application/json"
  },

  body: {
    address:  "UQXXXX...",
    amount:   "0.05"
  },

  success: "/done",
  error:   "/failed"

})
Standard Responses
200 Success Format
{
  "success": true,
  "data":    {}
}
4xx / 5xx Error
{
  "success": false,
  "error":   "Invalid API key"
}
